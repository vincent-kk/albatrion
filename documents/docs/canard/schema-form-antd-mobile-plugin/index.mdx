Ant Design Mobile UI plugin for `@canard/schema-form`.
Provides 10 touch-optimized form input components built on `antd-mobile ^5.0.0`.
Designed for mobile web applications ΓÇË switches and sliders are preferred over desktop-style inputs.

## Installation

```bash
yarn add @canard/schema-form-antd-mobile-plugin antd-mobile@>=5.0.0
```

### Peer Dependencies

```bash
yarn add dayjs@>=1 react@>=18 react-dom@>=18
```

Note: `antd-mobile` is a direct dependency of this plugin, not a peer dependency.

## Quick Setup

```ts
import { registerPlugin } from '@canard/schema-form';
import { plugin } from '@canard/schema-form-antd-mobile-plugin';

registerPlugin(plugin);
```

## Component Mapping

Components are selected automatically based on JSON Schema properties. Priority order matches the list below (top = highest priority).

| Component | type | format | formType | enum | Notes |
|---|---|---|---|---|---|
| `FormTypeInputBooleanSwitch` | `boolean` | ΓÇË | `"switch"` | ΓÇË | Switch toggle with custom labels |
| `FormTypeInputStringCheckbox` | `array` | ΓÇË | `"checkbox"` | items.enum | Checkbox group; items must be string schema |
| `FormTypeInputStringSwitch` | `string` | ΓÇË | `"switch"` | 2 values | Toggle between two enum values |
| `FormTypeInputRadioGroup` | `string` \| `number` \| `integer` | ΓÇË | `"radio"` or `"radiogroup"` | required | Radio group |
| `FormTypeInputArray` | `array` | ΓÇË | ΓÇË | ΓÇË | Dynamic list with add/remove |
| `FormTypeInputSlider` | `number` \| `integer` | `"slider"` | ΓÇË | ΓÇË | Slider; also supports range slider for numeric-item arrays |
| `FormTypeInputTextarea` | `string` | `"textarea"` or formType `"textarea"` | ΓÇË | ΓÇË | Multi-line textarea |
| `FormTypeInputString` | `string` | ΓÇË | ΓÇË | ΓÇË | Default text input (password supported via `format: "password"`) |
| `FormTypeInputNumber` | `number` \| `integer` | ΓÇË | ΓÇË | ΓÇË | Stepper component |
| `FormTypeInputBoolean` | `boolean` | ΓÇË | ΓÇË | ΓÇË | Checkbox with indeterminate state |

### Slider Range Support

`FormTypeInputSlider` activates for numeric range arrays:

```ts
// Single value slider
{ type: 'number', format: 'slider' }

// Range slider (array of two numbers)
{
  type: 'array',
  format: 'slider',          // or: items with numeric type + format:'slider'
  items: { type: 'number' }
}
```

## Schema Extension Properties

```ts
{
  formType?: 'switch' | 'radio' | 'radiogroup' | 'checkbox' | 'textarea',
  format?:   'slider' | 'textarea' | 'password',
}
```

## Usage Example

```tsx
import { Form } from '@canard/schema-form';

const schema = {
  type: 'object',
  properties: {
    name:    { type: 'string' },
    count:   { type: 'integer' },
    active:  { type: 'boolean', formType: 'switch' },
    rating:  { type: 'number', format: 'slider', minimum: 0, maximum: 5 },
    tags:    {
      type: 'array',
      formType: 'checkbox',
      items: { type: 'string', enum: ['js', 'ts', 'react'] },
    },
  },
};

<Form jsonSchema={schema} onChange={setValue} />
```

## Context: Custom Switch Labels

```tsx
// Pass checkboxLabels through context for BooleanSwitch
<Form
  jsonSchema={schema}
  context={{ checkboxLabels: { active: { true: 'On', false: 'Off' } } }}
/>
```
