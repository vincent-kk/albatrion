# @winglet

TypeScript utility libraries for common operations.

## Packages

| Package | Description |
|---------|-------------|
| [`@winglet/common-utils`](./common-utils/) | Array, object, Promise, and type utilities |
| [`@winglet/react-utils`](./react-utils/) | React hooks, HOCs, and Portal system |
| [`@winglet/json`](./json/) | JSON Pointer (RFC 6901) and JSON Patch (RFC 6902) |
| [`@winglet/json-schema`](./json-schema/) | JSON Schema traversal, filtering, and validation |
| [`@winglet/data-loader`](./data-loader/) | Async data batching and caching |
| [`@winglet/style-utils`](./style-utils/) | CSS class composition and scoped styles |
