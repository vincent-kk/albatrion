# @winglet

일반적인 작업을 위한 TypeScript 유틸리티 라이브러리 모음입니다.

## Packages

| Package | Description |
|---------|-------------|
| [`@winglet/common-utils`](./common-utils/) | 배열, 객체, Promise, 타입 유틸리티 |
| [`@winglet/react-utils`](./react-utils/) | React 훅, HOC, Portal 시스템 |
| [`@winglet/json`](./json/) | JSON Pointer (RFC 6901) 및 JSON Patch (RFC 6902) |
| [`@winglet/json-schema`](./json-schema/) | JSON Schema 순회, 필터링, 유효성 검사 |
| [`@winglet/data-loader`](./data-loader/) | 비동기 데이터 배칭 및 캐싱 |
| [`@winglet/style-utils`](./style-utils/) | CSS 클래스 조합 및 스코프 스타일 |
