# 자동 계산 필드

`computed.derived`를 사용하여 다른 필드의 값으로부터 자동으로 계산되는 필드를 보여줍니다. 가격, 수량, 세율, 할인율을 변경하면 최종 가격이 실시간으로 업데이트됩니다.

:::info 주요 기능
- `computed.derived` — 필드 값 자동 계산
- 부모 경로 참조(`../field`)를 사용한 복합 수식
- 의존성 변경 시 실시간 재계산
:::
